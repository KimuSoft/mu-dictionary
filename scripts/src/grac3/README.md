## 원신-키뮤사전 컨버터
### 개요
* reference: [게임물관리위원회 게임물 3.0 API](https://www.grac.or.kr/Game3.0/OpenAPIGuide.aspx)
* reference ID: `grac3`
 
### 사용법
* API 사용량 제한이 있으므로 1일 1회만 돌려야 합니다.
```bash
bun grac3
```

### 키뮤사전 지원
| 지원 | 속성               | 속성 설명    | 비고                                          |
|----|------------------|----------|---------------------------------------------|
| ✅  | `origin`         | 원어       |                                             |
| ✅  | `name`           | 단어명      |                                             |
| ✅  | `simplifiedName` | 단순화된 단어명 |                                             |
|    | `definition`     | 단어 정의    | `게임 '원신'에 등장하는 단어`                          |
| ✅  | `pos`            | 품사       | 모든 단어: 명사                                   |
| ✅  | `tags`           | 주제       | `게임`                                        |
| 🚫 | `thumbnail`      | 섬네일      |                                             |
| ✅  | `url`            | 링크       |  |

### 데이터 타입
* 해당 스크립트는 외부 오픈 API를 사용합니다.
